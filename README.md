####  Popupwindow的使用

<p>PopupWindow在Android.widget包下，项目中经常会使用到PopupWindow做菜单选项，&nbsp; PopupWindow这个类用来实现一个弹出框，可以使用任意布局的View作为其内容，这个弹出框是悬浮在当前activity之上的。</p>
